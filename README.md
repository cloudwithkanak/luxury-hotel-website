# 🏨 Royal Crest Hotel Website

A modern and responsive luxury hotel website built using HTML, CSS, and JavaScript. This project showcases a professional hotel website with multiple pages, responsive design, and a clean user interface. It is designed as a portfolio project and will later be deployed using Docker, GitHub Actions, and AWS EC2.

---

## 📌 Features

- Responsive Navigation Bar
- Home Page
- About Page
- Rooms Page
- Gallery Page
- Restaurant Page
- Booking Page
- Contact Page
- Booking Form Validation
- Contact Form
- Responsive Design
- Smooth Scrolling
- Scroll-to-Top Button
- Scroll Animations
- Professional Footer

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/luxury-hotel-website.git
```

### 2. Navigate to the Project

```bash
cd luxury-hotel-website
```

### 3. Open the Project

Open the project using Visual Studio Code.

### 4. Run the Website

Simply open:

```
index.html
```

in your web browser.
